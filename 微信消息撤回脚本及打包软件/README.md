##主要功能##
通过运行程序并通过微信扫码登录后，一旦有好友发来消息并撤回，就会收到来自程序的提醒，包括好友名、时间、撤回的具体消息。通知是通过文件助手来实现的。
关于退出：当你不想再继续使用时，通过手机微信退出网页版后，电脑桌面会弹出窗口，点击退出，相关进程便会退出。

##开发环境##
	Python3.6.4
	itchat1.3.10
	tkinter
	pyinstaller3.3.1
